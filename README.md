# LumaScale

브라우저에서 바로 실행되는 이미지 업스케일러입니다. 이미지 파일을 열고 2x, 3x, 4x 또는 `13500 x 10500 px` 같은 직접 출력 크기로 내보낼 수 있습니다.

LumaScale is a browser-based image upscaler. Open an image and export it at 2x, 3x, 4x, or a custom output size such as `13500 x 10500 px`.

## 기능

- PNG, JPG, WebP 이미지 열기
- 2x, 3x, 4x 업스케일
- `13500 x 10500 px` 프리셋 및 직접 W/H 입력
- 디테일 보정, 부드럽게, 픽셀 아트 모드
- 자동/선명하게 화질 개선 프리셋과 강도 조절
- 채우기, 전체, 늘이기 비율 처리
- 비교 슬라이더와 결과/원본 보기
- PNG, JPEG, WebP 내보내기

## Features

- Open PNG, JPG, and WebP images
- Upscale by 2x, 3x, or 4x
- `13500 x 10500 px` preset and custom width/height fields
- Detail Boost, Smooth, and Pixel Art modes
- Auto/Vivid quality enhancement presets with strength control
- Cover, Contain, and Stretch fit modes
- Compare slider with result/original views
- Export as PNG, JPEG, or WebP
- Korean/English language toggle

## 실행

`index.html` 파일을 브라우저에서 열면 됩니다.

## Run

Open `index.html` in a browser.
