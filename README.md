# LumaScale

브라우저에서 바로 실행되는 이미지 업스케일러입니다. 이미지 파일을 열고 2x, 3x, 4x 또는 `13500 x 10500 px` 같은 직접 출력 크기로 내보낼 수 있습니다.

## 기능

- PNG, JPG, WebP 이미지 열기
- 2x, 3x, 4x 업스케일
- `13500 x 10500 px` 프리셋 및 직접 W/H 입력
- 디테일 보정, 부드럽게, 픽셀 아트 모드
- 채우기, 전체, 늘이기 비율 처리
- 비교 슬라이더와 결과/원본 보기
- PNG, JPEG, WebP 내보내기

## 실행

`index.html` 파일을 브라우저에서 열면 됩니다.
