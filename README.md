# LumaScale

브라우저에서 바로 실행되는 사진 편집기입니다. 이미지를 자르고, 업스케일하고, 화질을 개선하고, 필터를 적용한 뒤 원하는 파일 형식으로 내보낼 수 있습니다.

LumaScale is a browser-based photo editor. Crop, upscale, enhance, filter, and export images in the format you need.

## 기능

- PNG, JPG, WebP 이미지 열기
- 퍼센트 기반 사진 자르기
- 2x, 3x, 4x 업스케일
- `1920 x 1080 px`와 `13500 x 10500 px` 프리셋 및 직접 W/H 입력
- 디테일 보정, 부드럽게, 픽셀 아트 모드
- 자동/선명하게 화질 개선 프리셋과 강도 조절
- 따뜻하게, 차갑게, 흑백, 시네마틱 필터
- 채우기, 전체, 늘이기 비율 처리
- 비교 슬라이더와 결과/원본 보기
- PNG, JPEG, WebP 내보내기

## Features

- Open PNG, JPG, and WebP images
- Crop images with percentage controls
- Upscale by 2x, 3x, or 4x
- `1920 x 1080 px` and `13500 x 10500 px` presets plus custom width/height fields
- Detail Boost, Smooth, and Pixel Art modes
- Auto/Vivid quality enhancement presets with strength control
- Warm, Cool, Mono, and Cinematic filters
- Cover, Contain, and Stretch fit modes
- Compare slider with result/original views
- Export as PNG, JPEG, or WebP
- Korean/English language toggle

## 실행

`index.html` 파일을 브라우저에서 열면 됩니다.

## Run

Open `index.html` in a browser.
